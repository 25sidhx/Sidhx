# 25sidhx - Portfolio & Resume Website

A modern, interactive portfolio and resume website built with **React**, **Vite**, and **Framer Motion**. Features a stunning cyberpunk-inspired UI with smooth animations and keyboard navigation.

## ✨ Features

- **Interactive Menu System** - Keyboard and mouse navigation with animated transitions
- **Responsive Design** - Optimized for desktop, tablet, and mobile devices  
- **Smooth Animations** - Framer Motion-powered interactions and page transitions
- **Modern Tech Stack** - React 19, Vite, React Router for fast performance
- **Cyberpunk Aesthetic** - Dark theme with neon accents and visual effects
- **Accessibility** - Keyboard shortcuts (Arrow keys + Enter) for full site navigation

## 🛠️ Tech Stack

- **Frontend**: React 19.2.4, React Router 7.14.0
- **Build Tool**: Vite 8.0.1
- **Animations**: Framer Motion 12.38.0
- **Styling**: CSS-in-JS with responsive design
- **Linting**: ESLint 9.39.4

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/25sidhx/Sidhx.git
cd Sidhx

# Install dependencies
npm install
```

### Development

```bash
# Start dev server (runs on http://localhost:5173)
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Run linter
npm lint
```

## 📋 Navigation

The portfolio features 5 main sections:

1. **About Me** - Personal introduction and background
2. **Resume** - Professional experience and skills
3. **GitHub Link** - Direct link to GitHub profile
4. **Socials** - Connect on various social platforms
5. **Side Projects** - Showcase of personal and hobby projects

### Keyboard Controls
- **↑** / **↓** - Navigate menu items
- **↵** - Open selected section
